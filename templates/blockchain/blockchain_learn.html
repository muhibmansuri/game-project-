What is a Block?

Think of it as a Digital Page in a notebook. Each page stores information like "Who sent money to whom". But it also contains a secret ID called a Hash.

Hashing (The Digital DNA)

Hashing converts any data into a fixed string of characters. If you change even 1 letter in the data, the entire Hash changes. This makes the data Tamper-Proof.

# Data: "Hello" -> Hash: 185f8...
# Data: "Hello." -> Hash: 75bc5... (Totally Different!)

Linking the Chain

Every block stores the Previous Hash of the block before it. This links them together. If you change Block #1, its Hash changes, which breaks the link to Block #2, and so on. This is why it's a Chain.

Mining (Proof of Work)

In our simulation, a valid block must have a Hash that starts with 00. Since we can't change the transactions, we change a random number called a Nonce millions of times until we find a matching Hash. This is Mining.

Genesis Block

The very first block in a blockchain is the Genesis Block. It has no previous block, so its Previous Hash is set to "0". It is the foundation of the entire chain.
